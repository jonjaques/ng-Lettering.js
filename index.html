Lettering.js

An AngularJS <span> injector
for radical Web Typography

This is a rework of the original Lettering.js which can be found here: https://github.com/davatron5000/Lettering.js.

<h1 data-lettering="letters">Some text which will inject spans around each character</h1> <div data-lettering="words">Text which will have spans injected around each word</div> <span data-lettering="lines">A slab of text </br> which will inject spans in place </br> of each br</span>
